# 🌸 Blossom & Bloom

A premium, elegant flower shop e-commerce UI built for a hackathon. Designed to feel like a real boutique brand — not a tutorial project.

Built with **HTML, CSS, JavaScript & React only** — no backend, no database.

> Deploy free on Vercel / Netlify

### Features

- **Modern UI/UX:** Minimalist design with soft pastels, serif headings, and glassmorphism navbar
- **Interactive Shop:** Product grid with hover-zoom images and animated "Best Seller" badges
- **Smart Filtering:** Filter bouquets by category (Wedding, Bouquets, Plants) with React state
- **Animated Cart Drawer:** Slide-out cart with add/remove, quantity update, and bounce animation on cart icon
- **Quick View Modal:** Click any flower for a Framer Motion popup with details
- **Persistent Cart:** Saves to LocalStorage — refresh and your cart stays
- **100% Responsive:** Mobile-first with Tailwind CSS

### Tech Stack

- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide-React
- **Data:** Mock JSON (`products.js`) + LocalStorage
